# Security Policy

## Reporting a Security Concern

If you believe you have discovered a security vulnerability in this project, please report it privately by contacting the project maintainers. We ask that you do not report security issues via public GitHub issues.

### Responsible Disclosure

- We aim to respond to security reports within 1-2 business days.
- We will provide updates throughout the remediation process.
- We will publicly disclose resolved security issues after they are patched, if relevant to the community.
- We ask for your patience as we investigate and address security concerns.

## Data Privacy

This application is designed for educational use. It stores progress data locally in the browser's `localStorage`. No personal information is transmitted to external servers without explicit user action.
