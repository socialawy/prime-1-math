# About Arabic

- The web application will absolutely need Arabic support, even though the core math lessons (like "Different Shapes" and "Addition by making 10") are taught in English. 

1. **Official Text & Ministry Approvals:** The book contains Arabic text right on the cover, such as **"المنهج الجديد"** (The New Curriculum) and **"تصريح وتحت إشراف"** (Authorized and under the supervision of). 
2. **Parental Guidance:** The book includes a section for "General notes for parents." In the Egyptian educational context (even in language schools where Math is taught in English), instructions meant for parents are often provided in Arabic to ensure clear communication.
3. **The Target Audience (Egypt):** Since this is an Egyptian curriculum app (El-Moasser), any overarching UI elements, help menus, or the "Parent/Teacher Dashboard" we discussed earlier will likely need an Arabic translation toggle to be fully accessible and user-friendly for the parents tracking the 7-year-old's progress.
